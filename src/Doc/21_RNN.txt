1. 第199 ~ 200行中:
=> Tokenizer()：
    Tokenizer 是 TensorFlow/Keras 提供的一個類，用來將文本數據轉換為數字序列。
    它會建立一個詞彙表，把每個詞映射到一個唯一的整數索引。
    簡單來說就是把文本數據轉為向量。
=> fit_on_texts 方法會根據 raw_text 中出現的所有詞來建立詞彙表，並將每個詞分配一個唯一的整數索引。
    這樣，文本中的每個詞就可以用對應的數字來表示，方便進一步的數據處理和模型訓練。

2. 第192 ~ 194行中:
=> np.hstack 函數在 NumPy 中的作用是沿著水平方向（即列方向）將多個數組拼接在一起。
    這裡的「水平」指的是將數組的列合併到一起，形成一個更寬的數組。

3. 第213 ~ 215行中:
=> pad_sequences 是 TensorFlow Keras 中的函數，用於對序列進行填充或裁剪，以便將所有序列調整為相同的長度。
    這在處理自然語言處理（NLP）和其他需要固定輸入長度的模型時非常有用。

4. 第204 ~ 207行中:
=> texts_to_sequences 是 TensorFlow Keras 中 Tokenizer 類別的一個方法，用於將文本序列轉換為數字序列。
    這個方法在 NLP 數據時非常有用，尤其是在將文本數據轉換為機器學習模型可以處理的數值格式時。

5.第197 ~ 215行中:
=> 實際上就是這整段程式碼就是
    A. 將文字轉成向量，(第199 ~ 200行) 
    B. 在將向量轉為數字向量，(第204 ~ 207行)
    C. 最後裁減為長度一致的數字向量(第213 ~ 215行)
